# ArduConsole
- ArduConsole is a ATMEGA328 based Open-Source Console designed to be modular, easy to modify and affordable.

## How is it affordable ?
- ArduConsole uses the ATMEGA328P MCU to run ArduOS (which is the official operating system for this console), LCD Character Screen for Display and a Joystick!
- And with only these components, we have successfully made a console with Graphics Support, Joystick Controls
